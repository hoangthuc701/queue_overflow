# Queue Overflow

# Setup
* Install Docker (https://docs.docker.com/get-docker/) 

#Clone and Run
## Clone this repo to your local machine 
> git clone https://github.com/hoangthuc701/queue_overflow.git
## Make file .env (see file .env.example)
## Run
> docker-compose up
## Stop
> docker-compose down
## Remove image
> docker image prune
